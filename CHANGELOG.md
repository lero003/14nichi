## [1.0.0] - 2026-07-24

初回 App Store 公開（iPhone / iPad）。ビルド `6`。

### 追加

- 同梱13状況・30記事のオフラインガイド（全文検索、カテゴリ・行動時期、お気に入り）
- 人数と7日・14日計画に基づく備蓄の一般的な目安、不足品目の買い物リスト
- 緊急カード（最小個人情報、端末内分離ストア、任意の Face ID / Touch ID / パスコード）
- 公式情報リンク集、項目選択と明示同意付きの PDF・印刷
- 読みやすさ設定、About、製品・サポート・プライバシーへのオンライン導線

### 対象外

- macOS 版（リポジトリに将来候補として残す。初回申請・商品ページには含めない）
- ネットワーク常時通信、アカウント、解析、広告

### 注意

- 記事の `approved` は公的一次情報との照合と編集確認を通過した意味であり、専門資格者による個別監修やあらゆる状況での正確性保証ではない
- 個人情報は端末内のみ。iCloud 同期や外部送信は行わない
