# 緊急カードに書いておくこと

緊急カードは、本人が伝えられないときに備える最小のメモです。多く書きすぎず、見られて困る情報とのバランスを取ります。

## 最初の5分ですること

- アプリの緊急カードを開き、空欄を確認する。
- 表示名、緊急連絡先、集合場所、避難予定場所、アレルギー、常用薬、注意メモのうち、本当に必要な最小項目から埋める。
- 紙に書く場合も、持ち歩き用と自宅保管用を分けるか決める。

## 今日中にすること

- 連絡先がつながるか、番号の書き間違いがないか確認する。
- 子どもや高齢者と一緒に、カードの場所と意味を共有する。
- 顔写真や詳細な病歴など、必要以上の個人情報を載せすぎない。
- 任意の表示ロック（Face ID / Touch ID など）を使うか決める。

## 明日以降にすること

- 引っ越し、転校、薬の変更のあとに必ず更新する。
- 家族全員分のカードや、ペットの預かり先情報も必要なら整える。
- PDF・印刷する場合は、出力項目と同意を確認し、置き場所を決める。

## やってはいけないこと

- パスワードやクレジットカード番号など、災害対応に不要な秘密情報を書かない。
- 血液型、詳細な診断名、保険証番号、本人確認書類番号など、このアプリの緊急カードで扱わない情報を注意メモへ書き足さない。
- カードを作っただけで、家族が場所を知らない状態にしない。
- クラウド同期やSNSへの投稿で、意図せず公開しない（このアプリは端末内保存）。

## 状況が悪化した場合

- すでに緊急事態なら、カードの未完成部分にこだわらず、わかる連絡先と症状を優先して伝える。
- 端末が使えない場合に備え、紙の最小メモも検討する。
